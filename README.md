# 阿里巴巴CRM客户采集插件 使用说明

## 📦 插件文件结构

```
alibaba-crm-scraper/
├── manifest.json       # 插件配置文件 (Manifest V3)
├── content.js          # 页面数据采集脚本
├── popup.html          # 插件弹窗界面
├── popup.js            # 弹窗交互逻辑 + 翻页控制
├── xlsx.min.js         # Excel 生成库（纯JS，无外部依赖）
└── icons/
    ├── icon16.png
    ├── icon32.png
    ├── icon48.png
    └── icon128.png
```

---

## 🚀 安装步骤

1. 打开 Chrome 浏览器，地址栏输入：`chrome://extensions/`
2. 右上角开启 **「开发者模式」**
3. 点击 **「加载已解压的扩展程序」**
4. 选择本文件夹（`alibaba-crm-scraper`）
5. 插件安装成功，工具栏会出现橙色图标 📊

---

## 📖 使用方法

### 基本使用
1. 登录阿里巴巴国际站，进入 **「客户」→「客户列表」** 页面
2. 点击浏览器工具栏的插件图标（橙色）
3. 选择采集方式：
   - **🚀 全部采集（自动翻页）**：自动逐页采集所有客户，支持千页级数据
   - **📋 仅采集当前页**：只采集当前显示的这一页，可多次点击累积数据
4. 采集完成后，点击 **📥 导出 Excel** 下载文件

### 注意事项
- 采集过程中请不要手动切换标签页
- 建议每次采集前先用筛选条件缩小范围
- 翻页间隔约 2.5 秒，请耐心等待
- 数据量大时（1000+页）建议分批采集，避免浏览器卡顿

## ❓ 常见问题

**Q: 点击全部采集后没有反应？**  
A: 请确认当前标签页是阿里巴巴CRM客户列表页，URL 应包含 `alibaba.com`

**Q: 采集到的数据为空？**  
A: 确认页面已完全加载，表格中有数据后再点击采集

**Q: 翻页采集卡住了？**  
A: 点击「停止」后，已采集的数据仍可导出。分批采集后合并即可

**Q: 导出的Excel打开乱码？**  
A: 用 Excel 或 WPS 直接打开，文件已是 .xlsx 格式，中文正常显示
